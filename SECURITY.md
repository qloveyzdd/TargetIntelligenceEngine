# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.0.x | Yes |
| < 1.0.0 | No |

## Reporting a Vulnerability

如果你发现了安全问题，请不要直接公开完整细节。

建议按这个顺序处理：

1. 优先使用 GitHub 的私密漏洞上报入口（如果仓库已启用）
2. 如果看不到私密上报入口，请联系维护者 `@qloveyzdd`
3. 在收到确认前，不要公开 PoC、利用方式或敏感环境信息

报告时尽量附上：

- 影响范围
- 复现步骤
- 可能风险
- 修复建议（如果有）

我们会尽快确认问题并安排处理。
