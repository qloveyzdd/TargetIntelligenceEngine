## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| persisted gaps -> relationship graph | persisted gap 如果被错误投影，会直接污染主图解释链 |
| graph click target -> focus metadata | 点击对象与高亮/解释 target 的映射必须稳定，否则面板会显示错对象 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-04 | Tampering | `buildRelationshipGraph()` | mitigate | 只从 `goal`、`dimensions`、`candidates`、persisted `gaps` 生成节点；显式拒绝 Evidence 与 Stage Goal 节点进入主图。 |
| T-05-05 | Denial of Service | dagre layout path | mitigate | 将布局收敛在纯函数 builder 中，只有 run 数据变化时重算，避免 UI 每次 render 重排。 |

</threat_model>
